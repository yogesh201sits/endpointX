# Backend Generator API

**Backend Generator** is an AI-powered backend code generator that allows developers to create fully functional backend servers instantly. Using **EndpointX**, you can generate, manage, and test backend endpoints automatically from a schema or natural language description.

This project leverages AI agents to automate code generation, file management, and sandbox execution, providing developers with a fast, seamless workflow from idea to running backend.

## Features
- AI-powered backend code generation
- Automatic deployment to sandbox environment
- Manage and test endpoints with **EndpointX**
- Supports multiple operations: GET, POST, DELETE, and more
- Easy integration with frontend applications
